# vs-ext-muted-boilerplate

A Visual Studio extension that visually mutes boilerplate code (telemetry calls, logging, method signatures, etc.) so it fades into the background while the meaningful logic stays prominent.

## Goals

- Mute boilerplate via styling (color, font size, font type) and optional auto-collapsed regions.
- User-configurable rules and styles, with sensible defaults.
- Hotkeys to toggle muting per category (telemetry, logging, signatures, ...).
- Architecture that allows shared core logic to be reused by a future VS Code extension.
